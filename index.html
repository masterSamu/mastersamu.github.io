Kids & Friends web sovellus

Web pohjainen maksullinen alusta missä perheet voivat verkostoitua toistensa kanssa. Mahdollisuus lähettää ja hyväksyä kaveripyyntöjä, lähettää yksityisviestejä ja ilmoittaa paikkakunnan tapahtumia.

Suunnittelin ja toteutin projektin freelancerina uudelle startup yritykselle. Progressiivinen web sovellus on toteutettu Reactilla ja Firebase backendillä.

Yrityksen nettisivut on tehty HTML ja Bootstrap yhdistelmällä.
